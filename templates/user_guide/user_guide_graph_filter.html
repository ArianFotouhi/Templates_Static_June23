Graph Filters

Regarding filters, the system provides users with various options to refine the displayed data based on their specific needs.

Client Filter

Users who have access to multiple clients can apply a client filter. This filter allows them to view data exclusively related to a particular client. By selecting a specific client, the system will display information and insights relevant only to that client.

Lounge Filter

In cases where clients have multiple lounges, users can further narrow down the data by selecting a particular lounge within the lounges associated with the accessed clients. This enables users to focus on the data related to a specific lounge.

Airport Filter

For users dealing with airport-related data, the airport filter allows them to choose a particular airport from the available options. Once applied, the system will present data specific to the selected airport, helping users to analyze and assess airport-related metrics effectively.

City and Country Filters

Users can also explore traffic data for specific cities or countries. By selecting the corresponding filters, the system will display data associated only with the chosen city or country, facilitating a better understanding of traffic patterns and trends in those locations.
