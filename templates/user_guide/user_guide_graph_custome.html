Graph Customization Tools

Regarding graph customization tools, users have the option to personalize their graphs using three main features: Graph Color Contrast, Plot Size, Curve Thickness and Plot Interval.

Graph Color Contrast is a valuable tool that enables users to control the extent of color variation in the graph, depending on whether there is an increase (resulting in a more bluish hue) or a decrease (resulting in a more reddish hue) in the plot. By default, the plot size is automatically set to match the dimensions of the user's monitor. However, users also have the flexibility to manually adjust the size of plots according to their preferences.

Moreover, increasing the curve thickness proves advantageous for enhancing visibility when dealing with smooth curves. On the other hand, reducing the thickness of a curve is beneficial for revealing finer details when dealing with curves that display a significant number of fluctuations.

Lastly, clarity meets convenience with plot interval tool. Smoothly it summarizes and aggregates results with the preferred interval (e.g., 5 days). This feature allows you to interpret graphs effortlessly, revealing patterns and insights with a glance.
